# StayFit - Personalized AI Health Coach

StayFit is a comprehensive, full-stack health and wellness platform built with modern technologies. It solves the challenge of personalizing fitness and nutrition advice through AI-driven insights tailored to each user's biological profile and fitness goals.

## 🚀 Key Features

-   **AI Health Coach**: Interactive 24/7 AI chat for fitness and health guidance.
-   **Personalized Diet Planner**: Generates 7-day meal plans based on metabolism and goals.
-   **Adaptive Workout Planner**: Creates custom exercise routines tailored to fitness levels.
-   **Dynamic Dashboard**: Visual monitoring of progress with glassmorphic UI elements.
-   **Secure Authentication**: JWT-based user security and profile management.
-   **Hyper-Local Insights**: Context-aware recommendations based on user demographics.

## 🛠️ Tech Stack

### Backend
-   **FastAPI**: High-performance Python framework for building APIs.
-   **PostgreSQL**: Robust relational database for health metrics and user data.
-   **SQLAlchemy**: Advanced ORM for database management.
-   **Gemini AI**: Powering the intelligent coaching and generation engines.
-   **Pydantic**: Data validation and architectural integrity.

### Frontend
-   **React + Vite**: Modern, high-speed frontend development ecosystem.
-   **TailwindCSS**: Utility-first styling for premium responsive design.
-   **Lucide React**: Beautiful, consistent iconography.
-   **Vite**: Blazing fast build tool.

## 📦 Getting Started

### Prerequisites
-   Python 3.10+
-   Node.js 18+
-   PostgreSQL

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/bd2711/StayFit_project.git
    cd StayFit_project
    ```

2.  **Backend Setup**:
    ```bash
    cd backend
    python -m venv venv
    ./venv/Scripts/activate # Windows
    pip install -r requirements.txt
    ```

3.  **Frontend Setup**:
    ```bash
    cd ../frontend
    npm install
    npm run dev
    ```
